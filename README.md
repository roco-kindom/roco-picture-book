# 洛克图鉴
Android原生实现的洛克王国图鉴, 遵循谷歌**Material3**设计. 您可以随时同步官网数据.

## 功能
- 官网数据同步
- 精灵信息查看 / 搜索
- 技能信息查看 / 搜索
- 道具信息查看
